# 黑光：进化

[English](README.md)

## 简介
《**黑光：进化**》（_**Blacklight: Evolution**_）是一个 Minecraft 模组，旨在重现电子游戏《**虐杀原形**》（_**PROTOTYPE**_）中的原型体各种能力和效果。

本模组尚处于非常早期的开发阶段，许多功能尚未实现，可能存在错误和性能问题。

此外，本模组是我的第一个 GitHub 项目和 Minecraft 模组，因此可以看作我的编程学习项目。
